# ServiceNow CAD

My working repository for the **ServiceNow Certified Application Developer (CAD)** track — study guides, PDI-ready code snippets, and running notes from the ServiceNow University bootcamp and beyond.

> **Exam at a glance:** 60 questions · 90 minutes · **70% to pass** · scenario-based.
> Read the guides, build everything in a free [Personal Developer Instance (PDI)](https://developer.servicenow.com/dev.do).

## Start here

Open **[`index.html`](index.html)** in a browser — it's the hub linking every guide, the blueprint, and the key external links. (GitHub won't render the interactive HTML; clone or download and open locally.)
