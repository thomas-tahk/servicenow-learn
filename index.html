ServiceNow Learn

My working repository for the whole ServiceNow journey — starting at CAD and building up the ladder. Platform fundamentals (reused by every cert) live separately from each certification's exam-specific material, so nothing is tied to one exam.

Start here: if you're new, read Platform Fundamentals first (it applies to every cert), then open the cert you're studying. Build everything in a free Personal Developer Instance.

How to trust & verify this material

This repo is study material — use it, but verify the load-bearing facts. The design keeps that lightweight:

SignalWhat it means for trust
"Verify against…" box (end of each guide)One link to the official ServiceNow source for that page's facts (blueprint, API). Click it to confirm — don't take weightings or method names on faith.
Runnable snippetsThe strongest verification: paste into your PDI and the platform itself confirms the code is right. If it runs, it's right.
QuizzesEach guide's quiz answers restate the key facts with explanations — a quick self-check that you (and the material) have it right.
Teaching vs. factTables, API names, and exam weightings are claims to verify against official docs. Framing like "start here" or "prefer the lower-code tool" is my teaching judgment, not gospel.
Avoiding overload Material is added one guide at a time, newest noted in the repo's commit history. You don't need to read it all at once — open the cert you're on, read its linked fundamentals sections, and skip the rest until you need it.
