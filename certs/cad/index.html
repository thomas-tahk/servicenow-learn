Certified Application Developer (CAD)

The exam-specific layer: blueprint, domain weightings, and a progress tracker. The skills themselves live in Platform Fundamentals — this page is just the map of what CAD tests and how heavily.

Exam at a glance: 60 questions · 90 minutes · 70% to pass · scenario-based. Build everything in a free Personal Developer Instance — not your work instance.
How CAD relates to the rest of the ladder CAD assumes admin fundamentals (the CSA level). If the App-Engine/admin basics feel shaky, the CSA material fills exactly that gap. After CAD come the specialist (CIS/CAS) and architect (CTA/CMA) tracks — all built on the same platform fundamentals.

Which CAD domain carries the most weight — and is the biggest gap for someone coming from daily use?

Managing Applications (scoped apps, Studio, update sets, versioning) is 25% and almost entirely new to a user who hasn't built before. Start there.

In a scoped application named "Asset Tracker," what do its table and field names look like?

A scoped app gets its own namespace (x_<company>_<app>_…), keeping its tables, fields, and scripts isolated from the global platform. That isolation is the whole point of the 25% Managing Applications domain.
